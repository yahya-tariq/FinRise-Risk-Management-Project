# FinRise Digital Finance Ltd. – Risk Management Project
📌 Overview

This project was completed as part of the Financial Risk Analytics and Derivatives (AF-4012) course. The objective is to analyze and manage financial risks in a fintech company, FinRise Digital Finance Ltd., using both theoretical and quantitative approaches.

The project is divided into three main parts:

Risk Identification and Analysis
Portfolio Construction and Optimization
Credit Risk Modelling

🔍 Part 1: Risk Management Report

This section focuses on identifying and analyzing key risks faced by FinRise, including:

Credit Risk
Market Risk
Liquidity Risk
Operational Risk
Model Risk
Information Asymmetry

It also evaluates the impact of fintech technologies on risk and addresses key conceptual questions related to digital lending and automated credit systems.

📊 Part 2: Portfolio Construction

This section involves building an investment portfolio using:

Five stocks listed on the Pakistan Stock Exchange (PSX)
Treasury Bills
Bank Deposits

Key tasks include:

Data collection and preprocessing
Return calculation
Portfolio allocation
Risk (standard deviation) estimation
Sharpe ratio calculation
Portfolio optimization

All analysis is performed using Python.

🤖 Part 3: Credit Risk Modelling

This section focuses on predicting borrower default using machine learning techniques.

Key steps:

Data cleaning and preprocessing
Logistic regression modelling
Model evaluation (Confusion Matrix, ROC Curve, AUC)
Probability of Default (PD) threshold optimization
Financial impact analysis
