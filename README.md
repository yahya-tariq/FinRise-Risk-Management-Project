# FinRise Digital Finance Ltd. – Risk Management Project

## 📌 Overview
This project was completed as part of the **Financial Risk Analytics and Derivatives (AF-4012)** course. It focuses on analyzing and managing financial risks in a fintech company, *FinRise Digital Finance Ltd.*, using both theoretical concepts and quantitative techniques.

The project is divided into three main parts:
1. Risk Identification and Analysis  
2. Portfolio Construction and Optimization  
3. Credit Risk Modelling

## 🔍 Part 1: Risk Management Report
This section focuses on identifying and analyzing the major risks faced by FinRise, including:

- Credit Risk  
- Market Risk  
- Liquidity Risk  
- Operational Risk  
- Model Risk  
- Information Asymmetry  

It also explains how fintech technologies influence these risks and addresses key questions related to digital lending and automated credit scoring systems.

---

## 📊 Part 2: Portfolio Construction
This section involves building and analyzing an investment portfolio consisting of:

- Five stocks listed on the Pakistan Stock Exchange (PSX)  
- Treasury Bills  
- Bank Deposits  

Key tasks include:

- Data collection and preprocessing  
- Daily return calculation  
- Portfolio allocation  
- Risk (standard deviation) estimation  
- Sharpe ratio calculation  
- Portfolio optimization  

All analysis is performed using Python.

---

## 🤖 Part 3: Credit Risk Modelling
This section focuses on predicting borrower default using machine learning techniques.

Key steps include:

- Data cleaning and preprocessing  
- Logistic regression modelling  
- Model evaluation (Confusion Matrix, ROC Curve, AUC)  
- Probability of Default (PD) threshold optimization  
- Financial impact analysis
